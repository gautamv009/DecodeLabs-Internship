# DecodeLabs Data Analytics Internship

## Overview

This repository contains the projects completed as part of the DecodeLabs Data Analytics Internship Program. Each project focuses on a different aspect of data analytics, ranging from exploratory data analysis and machine learning to SQL-based business intelligence.

The internship emphasizes hands-on learning through real-world datasets and practical problem-solving techniques used in the data analytics industry.

---

## Internship Projects

### Week 1 - Exploratory Data Analysis (EDA)

#### Objective
Perform data cleaning, preprocessing, and exploratory data analysis to understand patterns, trends, and relationships within the dataset.

#### Key Tasks
- Data loading and inspection
- Handling missing values
- Duplicate detection and removal
- Statistical analysis
- Data visualization
- Feature understanding
- Insight generation

#### Skills Gained
- Data Cleaning
- Exploratory Data Analysis
- Data Visualization
- Statistical Interpretation
- Python Data Analysis

---

### Week 2 - Customer Churn Prediction

#### Objective
Build a machine learning model to predict whether a customer is likely to leave a service based on historical customer data.

#### Key Tasks
- Data preprocessing
- Feature engineering
- Feature selection
- Model training
- Model evaluation
- Performance analysis

#### Algorithms & Techniques
- Logistic Regression
- Data Scaling
- Train-Test Split
- Classification Metrics

#### Skills Gained
- Machine Learning
- Predictive Analytics
- Classification Models
- Model Evaluation
- Feature Engineering

---

### Week 3 - SQL Data Analysis

#### Objective
Analyze an E-commerce dataset using SQL queries to extract meaningful business insights and support decision-making.

#### Key Tasks
- Database creation using SQLite
- Data querying using SQL
- Data filtering
- Sorting and ranking
- Aggregation analysis
- Business insight generation

#### SQL Concepts Used
- SELECT
- DISTINCT
- WHERE
- ORDER BY
- GROUP BY
- COUNT()
- SUM()
- AVG()
- MAX()
- MIN()

#### Skills Gained
- SQL Query Writing
- Data Aggregation
- Business Intelligence
- Data Filtering and Sorting
- SQL-Based Reporting
