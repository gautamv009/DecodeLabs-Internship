# Exploratory Data Analysis (EDA)

This project was completed as part of my Data Analytics Internship at DecodeLabs.

The objective of this project was to perform Exploratory Data Analysis (EDA) on an e-commerce dataset to understand customer purchasing behavior, identify trends, analyze distributions, detect outliers, and generate meaningful insights from the data.

Through this project, I explored the dataset using descriptive statistics and visualizations to better understand patterns hidden within the data and support data-driven decision making.

## Dataset Information

- Total Records: 1200
- Total Columns: 14
- Dataset Type: E-commerce Transactions
- Time Period: January 2023 to June 2025

## Analysis Performed

### Dataset Exploration
- Examined dataset structure and data types
- Reviewed column information and record counts
- Analyzed the time range covered by the dataset

### Descriptive Statistics
- Calculated count, mean, median, minimum, and maximum values
- Analyzed customer purchasing patterns
- Evaluated transaction value distributions

### Product Analysis
- Identified the most frequently purchased products
- Compared purchase frequency across product categories
- Analyzed average revenue generated by each product

### Order Analysis
- Examined order status distribution
- Analyzed completed, cancelled, pending, and returned orders

### Payment Method Analysis
- Studied customer payment preferences
- Compared usage of different payment methods

### Referral Source Analysis
- Investigated customer acquisition channels
- Compared traffic generated from different referral sources

### Distribution and Outlier Analysis
- Visualized transaction value distributions
- Detected unusual transactions using boxplots and the IQR method

### Correlation Analysis
- Examined relationships between numerical variables
- Identified factors influencing transaction value

## Tools Used

## Key Findings

- Printers were the most frequently purchased product category.
- The average order value was approximately ₹1054.
- Online payments were the most commonly used payment method.
- Instagram generated the highest number of customer referrals.
- Laptop purchases produced the highest average transaction value.
- A small number of high-value transactions influenced the overall revenue distribution.
- TotalPrice showed a strong relationship with Quantity and UnitPrice.
