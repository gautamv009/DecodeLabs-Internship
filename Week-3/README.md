# Week 3 - SQL Data Analysis

## Project Overview
This project was completed as part of the DecodeLabs Data Analytics Internship Program (Week 3).

The objective of this project is to perform SQL-based analysis on an E-commerce dataset and extract meaningful business insights using various SQL operations. The project demonstrates the use of SQL for querying, filtering, sorting, grouping, and aggregating data to support data-driven decision-making.

---

## Objectives

- Perform data analysis using SQL queries
- Retrieve and filter records using SELECT and WHERE clauses
- Sort data using ORDER BY
- Group data using GROUP BY
- Apply aggregate functions such as COUNT(), SUM(), AVG(), MAX(), and MIN()
- Generate business insights from transactional data

---

## Dataset Information

The dataset contains E-commerce order details including:

- Order ID

- Quantity Purchased
